===============================================
Fiona: access to simple geospatial feature data
===============================================

Fiona streams simple feature data to and from GIS formats like GeoPackage and
Shapefile. Fiona can read and write real-world data using multi-layered GIS
formats, zipped and in-memory virtual file systems, from files on your hard
drive or in cloud storage. This project includes Python modules and a command
line interface (CLI).
